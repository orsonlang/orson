GNU C Sources for the Orson Library

James B. Moen
January 27, 2018

Take a header into the sea.

[SEN 2002]

This directory contains many small GNU C source files [STA 2008] that help implement parts of the Orson library [MOE 2017]. Most have only include directives that insert header files from the GNU C library [LOO 2014]. However, the file widths.c initializes some large arrays, which can’t be done efficiently by the current version of Orson.

These files are available for free, under the terms of the GNU General Public License (GPL) version 3, so they may be used by free Orson programs. People bothered by this are invited to make their Orson programs free, or to write their own libraries. The file widths.c has the usual GNU copyright notice, but the other C sources do not, because it seems silly to put such notices in files with only one or two lines.
